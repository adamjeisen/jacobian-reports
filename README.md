# jacobian-reports

Auto-generated JacobianODE sweep analysis reports.

Layout: `<wandb_project>/<wandb_group>/` — each group directory contains
`report.html`, `report.md`, `figures/`, `metrics.json`, `context.json`
(sentinel snapshot), `run_analytics.log`, and any Claude-written
`discussion.md`.

Pushed automatically by `engaging-analyze` after each successful
analysis run.
